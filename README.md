# Bloom

> Bridging the educational gap by utilizing unused spaces for skill building events.

---

## Problem

- Institutional/corporate spaces go unused during weekends and festive times
- Students wants to give back to community via **social initiatives** (sometimes due to academic requirement)
- Learners lack access to **structured opportunities**, esecially under privileged

---

## Solution

A platform where:

- Institutions list **venues** (location, timing, facilities)
- Mentors create **events/workshops**
- Learners **discover & apply**

---

## Workflow

### 👨‍🏫 Mentor

- Sign up (Email Auth)
- Select venue **or** create online event
- Add:
  - Name, Description
  - Tags (Domain)
  - Capacity
  - Date & Time
  - Venue / Meet Link
- Publish event

### 👩‍🎓 Learner

- Sign up
- Browse events
- Filter via tags
- Apply & attend

---

## Features

- Authentication
- Venue Listing
- Event Creation
- Tag-based Filtering
- QR Attendance
- Online + Offline Events
